# Gemini Bug Hunter - Official Website

A minimalist, cyberpunk-themed landing page for Gemini Bug Hunter.

## Features
- **Matrix Rain Animation**: Custom canvas-based ASCII rain with curved/wind effects.
- **Scroll Interactions**: Elements fade in and slide up as you scroll.
- **Minimalist Design**: High-contrast Black & White aesthetic.
- **Responsive**: Fully responsive flex/grid layout.

## Tech Stack
- **HTML5 Canvas** (No external libraries)
- **Vanilla CSS3** (Animations, Keyframes)
- **Vanilla JavaScript** (Intersection Observer)
- **Vite** (Build tool)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Build

```bash
# Build for production
npm run build

# Preview build
npm run preview
```
